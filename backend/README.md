# 🧠 Backend Documentation

هذا المجلد يحتوي على "العقل المدبر" للمشروع (Server-Side Logic).

## 📂 محتويات المجلد:

### 1. [`app.py`](./app.py)
**نوع الملف:** Python Script (FastAPI)
**الوظيفة:**
هو السيرفر الرئيسي للمشروع. يقوم بالمهام التالية:
- **إدارة الـ API:** يستقبل طلبات الشات والصوت من الواجهة.
- **تحميل الموديلات:** يقوم بتحميل موديل `Whisper` (للصوت) وموديل `AraGPT2` (للنصوص) عند التشغيل.
- **كشف اللهجة:** يحتوي على خوارزمية `detect_dialect` التي تحلل النص وتحدد اللهجة المصرية.
- **إدارة الجلسات:** يحفظ تاريخ المحادثة لكل مستخدم (Memory).
- **توليد الصوت (TTS):** يحول رد البوت النصي إلى ملف صوتي ويرسله للواجهة.

### 2. `system_prompt.py`
**نوع الملف:** Python Script
**الوظيفة:**
ملف إعدادات الشخصية (Configuration). يحتوي على:
- **System Prompt:** النص الذي يوجه الذكاء الاصطناعي (أنت مساعد خدمة عملاء...).
- **القواعد:** تعليمات صارمة للبوت (مثل: الرد بنفس اللهجة، التعامل بتهذيب).
