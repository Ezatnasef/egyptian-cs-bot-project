# 🎨 Frontend Documentation

هذا المجلد يحتوي على واجهة المستخدم (User Interface) التي تظهر في المتصفح.

## 📂 محتويات المجلد:

### 1. [`index.html`](./index.html)
**نوع الملف:** HTML5 (+ Embedded CSS/JS)
**الوظيفة:**
الصفحة الوحيدة للتطبيق (Single Page Application). تحتوي على:
- **التصميم (UI):** شكل الشات، الأزرار، والألوان (CSS).
- **تسجيل الصوت:** كود JavaScript للوصول للميكروفون وتسجيل صوت المستخدم.
- **الاتصال بالسيرفر:** دوال `fetch` لإرسال الصوت والنص لـ `backend/app.py`.
- **تشغيل الصوت:** مشغل صوتي لتشغيل رد البوت (TTS) تلقائياً.
- **عرض الحالة:** إظهار اللهجة المكتشفة والمشاعر الحالية.
